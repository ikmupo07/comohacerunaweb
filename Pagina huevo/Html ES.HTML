HTML

El lenguaje de marcado de hipertexto o HTML es el lenguaje de marcado estándar para documentos diseñados para mostrarse en un navegador web. Puede ser asistido por tecnologías como hojas de estilo en cascada y lenguajes de secuencias de comandos como JavaScript.

Los beneficios del  HTML

Experiencia de usuario.
Debido a que HTML5 ofrece una amplia variedad de herramientas, puede diseñar y presentar sitios web y aplicaciones que superen las expectativas del usuario. En consecuencia, aumentar las visitas y brindar una buena experiencia.

Código más limpio.
Para los desarrolladores web, tratar de realizar cambios en el código desordenado puede ser una situación aterradora. Sin embargo, HTML5 se ha creado para ofrecer un código más limpio, lo que soluciona el problema tan frecuente de tener que hacer cambios en un sitio web.

Aplicaciones para programar

Sublime Text es un editor de código fuente multiplataforma shareware. Admite de forma nativa muchos lenguajes de programación y lenguajes de marcado. Los usuarios pueden ampliar su funcionalidad con complementos, generalmente creados por la comunidad y mantenidos bajo licencias de software libre.

Replit, anteriormente Repl.it, es una empresa nueva con sede en San Francisco y un entorno de desarrollo integrado en línea. Replit es un software como servicio que permite a los usuarios crear proyectos en línea y escribir código en muchos idiomas admitidos.

Visual Studio Code, también conocido comúnmente como VS Code, es un editor de código fuente creado por Microsoft con Electron Framework, para Windows, Linux y macOS. Las características incluyen soporte para depuración, resaltado de sintaxis, finalización de código inteligente, fragmentos, refactorización de código y Git integrado.
