CSS

CSS es el lenguaje para describir la presentación de páginas web, incluidos los colores, el diseño y las fuentes. Permite adaptar la presentación a diferentes tipos de dispositivos, como pantallas grandes, pantallas pequeñas o impresoras. CSS es independiente de HTML y se puede utilizar con cualquier lenguaje de marcado basado en XML.

¿Cómo funciona CSS?

CSS funciona como un complemento a la información que forma parte de un sitio web. Mientras que el código HTML incluye todos los datos, el código CSS se encarga de formatearlos y presentarlos visualmente a través de un navegador.

Al acceder a un sitio web, el navegador debe rastrear la información contenida en el HTML y traducirla a un DOM (o modelo de objeto de documento). Estos objetos deben coincidir con los bloques de código correspondientes en CSS para que se les aplique el estilo elegido y aparezcan en el formato asignado a la computadora..

Para que sirve el CSS

CSS se puede usar para estilos de texto muy básicos, como cambiar el color y el tamaño de los encabezados y enlaces. Se puede usar para crear un diseño, como convertir una columna de texto en una composición (en-US) con un área de contenido principal y una barra lateral para información relacionada. Incluso se puede utilizar para crear efectos de animación.

¿Qué es CSS?

CSS significa hojas de estilo en cascada. Básicamente, es un lenguaje que gestiona el diseño y la presentación de las páginas web, es decir, cómo se ven cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de las páginas.

Se denominan hojas de estilo "en cascada" porque puede tener varias hojas y una de ellas con las propiedades heredadas (o "en cascada") de otras.

Para muchas personas, una simple plantilla de blog es suficiente. Aun así, cuando quieras personalizar la apariencia de un sitio necesitarás implementar CSS que, junto con un buen CMS, te ayudarán a aumentar el alcance de tu contenido.

El CSS externo

El CSS externo existe en su propio archivo. Este archivo está vinculado a un documento HTML con un <enlace> etiqueta. Este es el método más utilizado para agregar CSS a HTML, ya que una hoja de estilo externa puede dictar el estilo de diferentes documentos HTML. Esto permite a los desarrolladores realizar cambios en todo el sitio con un solo archivo CSS.

El CSS en línea

El CSS en línea se coloca dentro de una etiqueta HTML para cambiar el estilo de un elemento específico. La sintaxis en línea es un poco diferente de lo que hemos visto: la declaración se escribe como el valor del atributo de estilo.
Aunque es posible, usar este método viola la práctica de separar el estilo del contenido, por lo que no se recomienda su uso . CSS en línea es ineficiente para la programación y más difícil de entender que los otros dos. Aun así, es bueno que lo conozcas para que puedas identificarlo, si se da el caso. 

El CSS Interno

El CSS interno es un código CSS incrustado en un documento HTML. Está escrito dentro del <style> que se coloca en el elemento <head> sección.
El CSS interno se puede utilizar para algunos proyectos web pequeños y páginas individuales con su propio estilo. En estos casos será más fácil guardar todo tu código en un mismo archivo y no en dos.
