jAVASCRIPT

JavaScript (JS) es un lenguaje de programación ligero, interpretado o compilado justo a tiempo con funciones de primera clase. Si bien es más conocido como el lenguaje de secuencias de comandos para páginas web, muchos entornos que no son de navegador también lo usan, como Node.js, Apache CouchDB y Adobe Acrobat.

¿Qué es realmente JavaScript?

JavaScript es un lenguaje de programación robusto que puede aplicarse a un documento HTML y usarse para crear interactividad dinámica en sitios web. Fue inventado por Brendan Eich, cofundador del proyecto Mozilla, la Fundación Mozilla y Mozilla Corporation.

Puedes hacer casi cualquier cosa con JavaScript. Puede comenzar con pequeñas cosas como carruseles, galerías de imágenes, diseños fluctuantes y respuestas a los clics de botones. ¡Con más experiencia, podrá crear juegos, animaciones 2D y gráficos 3D, aplicaciones integradas basadas en bases de datos y mucho más!

JavaScript en sí mismo es bastante compacto pero muy flexible, y los desarrolladores han escrito una gran cantidad de herramientas además del lenguaje principal de JavaScript, desbloqueando una gran cantidad de funciones adicionales con un mínimo esfuerzo.

¿Qué está haciendo JavaScript en tu página?

Aquí comenzaremos a ver algo de código y, mientras estamos en eso, exploraremos qué sucede realmente cuando ejecutas JavaScript en tu página.

Recapitulemos brevemente la historia de lo que sucede cuando carga una página web en un navegador. Cuando carga una página web en su navegador, está ejecutando su código (HTML, CSS y JavaScript) dentro de un entorno de ejecución (la pestaña del navegador). Esto es como una fábrica que toma materias primas (el código) y genera un producto (la página web).

¿Qué beneficios ofrece JavaScript?

Fácil de aprender y usar

La sintaxis de JavaScript se basa en el lenguaje de programación Java y es fácil de aprender y codificar. Los desarrolladores utilizan JavaScript en casi cualquier sitio web y aplicación móvil para la creación de secuencias de comandos del lado del cliente. Node.js ha ganado popularidad para escribir código de back-end durante la última década. Varias de las principales plataformas de transmisión y video se han codificado en Node.js.

Obtenga independencia de la plataforma

A diferencia de otros lenguajes de programación, JavaScript se puede incrustar en cualquier página web y usarse con muchos otros lenguajes y marcos de desarrollo web. Una vez que lo haya escrito, puede ejecutar código JavaScript en cualquier máquina. De esta forma, la tecnología JavaScript permite que el desarrollo de aplicaciones sea independiente de la plataforma.

Reducir la carga del servidor

Puede usar JavaScript para reducir la carga del servidor y la congestión de la red al permitir ejecutar operaciones lógicas y hacer gran parte del trabajo del servidor en el propio cliente. Por ejemplo, considere el proceso de completar un formulario de registro. JavaScript comprueba rápidamente si ha introducido un número de 10 dígitos para el campo "móvil". Si estas solicitudes se enviaran al servidor, la página se recargaría por cada error, lo que haría que el proceso de registro fuera muy lento y tedioso.

Mejorar la interfaz de usuario

JavaScript crea sitios web sofisticados que facilitan la búsqueda y el procesamiento de información compleja. Los desarrolladores usan JavaScript para ampliar la funcionalidad y la legibilidad, así como para hacer que la interacción del usuario con el sitio web sea más eficiente.
